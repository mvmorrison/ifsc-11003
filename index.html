Thoughts on Progressive Enhancement

Progressive enhancement is a web development approach that focuses on a basic, functional, and clean experience so every user can access the core, basic content. Afterwards, adding CSS for better presentation and JavaScript for more advanced users and option interactions, and makes sure the website remains useable across a wide range of devices. This is imporant in modern web because it improves accessibility and supoprts a wide range of devices while making sure the site stays useable if the scripts fail.

Mars Rover Project Revesion and Reflection

The Mars Rover Project is an assignment we recieved in place of a final exam, to which we had to build and code a website over either Nasa Rovers, or picking our own subject. Looking back over the course of the semester, I believe that I did well on color picking for the website. I tried to go for more space adjacent and Mars associeted colors for the color scheme, to which I believe I did well on. I had some little difficulies and kinks to work out on this final, to which was a little frustrating, but after trial and error I figured it out. If I could start the semester over, I'd try to be more ontop of everything. The feedback I recieved on the final wasn't much, only two things; to make the NASA logo bigger and to change the color of a text from red to orange. Taking those simple tweaks and feedback into consideration, I decided to go ahead and follow their feedback and make the changes suggested.
